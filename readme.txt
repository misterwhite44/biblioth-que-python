Il faut éxécuter le fichier evaltest.py pour ensuite avoir accès à l'interface utilisateur.

Quand on éxécute, il y a 8 options qui s'offrent à vous.

Choisissez ce que vous voulez faire en entrant le chiffre correspondant à l'option.

Pour sauvegarder les données dans le fichier json, il vous faudra entrer l'option 8.
et Ensuite, les données seront stockées dans le fichier json qui est nommé bibliothèque.json.

Le fichier est déja créer, du coup il n'y aura pas besoin de le créer(même si il se créer automatiquement après la sauvegarde).
